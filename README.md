# MedicOS

Sistema SaaS para gestión clínica.

Proyecto desarrollado con arquitectura backend basada en:

- NestJS
- Prisma ORM
- PostgreSQL

Estructura inicial del proyecto:

MedicOS
├── backend
├── README.md

Repositorio inicializado en T0.1 dentro de:

/home/btx/Documentos/MedicOS

El backend se encuentra en:

/home/btx/Documentos/MedicOS/backend

El control de versiones del proyecto pertenece al repositorio ubicado en la raíz del proyecto.

Este repositorio contiene la infraestructura base del sistema MedicOS y su evolución a través de las distintas fases del proyecto.
